# Introduction to AgentLayerWeb

AgentLayerWeb (v0.1) is the **Agent Experience (AX)** framework for building web applications that are natively optimized for both humans and AI agents. It is a framework-agnostic, client-side TypeScript library that adds a semantic layer on top of standard visual components.

AgentLayerWeb does **not** replace HTML, React, or your UI library. Instead, it adds a lightweight, zero-overhead semantic AX layer.

```
Application
├── UI Layer          (Optimized UX for humans)
└── AgentLayerWeb     (Optimized AX for AI agents)
```

By annotating your markup, humans continue using your visual interface normally, while AI agents (like Claude, GPT-4, or other WebMCP browser agents) gain a structured understanding of page intent, available actions, form requirements, and active workflows. This ensures websites are usable and efficient for both audiences.
